## Reporting A Vulnerability
  To Report,Go To A ~Pull Request~ Branch Called Vulnerability Report.Then,Create An Issue In **That Pull Request**.I Will Review It And Try Fix It.
  Private Report Method:
1. On GitHub, navigate to the main page of the repository.
2. Under the repository name, click  Security. If you cannot see the "Security" tab, select the dropdown menu, and then click Security.
3. Fill in the advisory details form.
4. At the bottom of the form, click Submit report. GitHub will display a message letting you know that maintainers have been notified and that      you have a pending credit for this security advisory.
5. When the report is submitted, GitHub automatically adds the reporter of the vulnerability as a collaborator and as a credited user on the     
   proposed advisory.
6. Optionally, click Start a temporary private fork if you want to start to fix the issue. Note that only the repository maintainer can merge       changes from that private fork into the parent repository.
